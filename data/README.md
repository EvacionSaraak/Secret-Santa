# Data Files

This directory contains the data files for the Secret Santa application.

## Files

- **participants.txt** - List of all participant names (one per line)
  - Edit this file to add/remove participants
  - Names are automatically converted to Camel Case
  - Total boxes equals total participants
  
- **secret-santa-state.json** - Example/initial state structure
  - Shows the JSON format for box assignments
  - Each box has `picker` and `assigned` fields
  
- **sample-event.json** - Sample event data for reference
